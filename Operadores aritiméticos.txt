Vamos falar sobre alguns operadores aritiméticos

+ Adição
- Subtração
* Multiplicação
/ DIvisão
** Exponenciação
// Divisão Inteira
% Resto da Divisão

Exemplos

Adição:
5 + 2 = 7

Subtração:
5 - 2 = 3

Multiplicação
5 * 2 = 10

Divisão:
5 / 2 = 2.5

Exponenciação:
5 ** 2 = 25

Divisão Inteira:
5 // 2 = 2

Resto da Divisão:
5 % 2 = 1

Eu posso também manipular a quantidade de casas decimais em um valor flutuante
por exemplo:

n1 = int(input('Digite um numero: ')) ####4, 3
n2 = int(input('Digite outro numero: '))
d = n1 / n2
print('A divisão vale {}'.format(d))

ele mostraria:
1.3333333333333

porém eu poderia fazer assim:

n1 = int(input('Digite um numero: ')) ####4, 3
n2 = int(input('Digite outro numero: '))
d = n1 / n2
print('A divisão vale {:.3f}'.format(d))

ele mostraria:
1.333

No caso aquele 3 f seria 3 casas flutuantes

se eu tivesse dois prints por exemplo:

n1 = int(input('Digite um numero: ')) ####4, 3
n2 = int(input('Digite outro numero: '))
d = n1 / n2
s = n1 + n2
print('A divisão vale {:.3f}'.format(d))
print('A soma vale {}'.format(s))

ele mostraria:
A divisão vale 1.333
a soma Vale 7



Pra ele não quebrar o print eu poderia fazer:

n1 = int(input('Digite um numero: ')) ####4, 3
n2 = int(input('Digite outro numero: '))
d = n1 / n2
s = n1 + n2
print('A divisão vale {:.3f}'.format(d), end=' ' )
print('A soma vale {}'.format(s))

ele mostraria:
A divisão vale 1.333  a soma Vale 7


ou se eu quiser quebrar a linha do print eu poderia faze

n1 = int(input('Digite um numero: ')) ####4, 3
n2 = int(input('Digite outro numero: '))
d = n1 / n2
s = n1 + n2
print('A divisão vale \n {:.3f}'.format(d), end=' ' )
print('A soma vale \n {}'.format(s))

ele mostraria:

1.333  a soma Vale
